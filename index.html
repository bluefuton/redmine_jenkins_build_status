A plugin for Redmine.

Installation

Clone into Redmine's vendor/plugins folder.

Inside the plugin's config folder, copy jenkins.example.yml to jenkins.yml and add your Jenkins JSON API URL to it. For example:

production:
  api_url: http://jenkins.example.com/api/json

Conventions

The project identifier used in Jenkins must match the project identifier used in Redmine.
